# KarteikartenApp

Eine **Java Swing Desktop-Anwendung** zum Erstellen und Lernen mit digitalen Karteikarten, mit Anbindung an eine **MySQL-Datenbank**.

## Hintergrund

Dieses Projekt ist zu dem Zeitpunkt entstanden, als ich angefangen habe, mich mit **objektorientierter Programmierung** auseinanderzusetzen. Das Ziel war nicht primär eine fertige, produktionsreife App, sondern das **Verständnis von OOP-Konzepten in der Praxis** – Klassen, Verantwortlichkeiten, Datenmodellierung, Persistenz.

Die Umsetzung erfolgte mit Unterstützung einer KI. Statt fertigen Code zu übernehmen, habe ich gezielt nachgefragt, recherchiert und versucht zu verstehen, *warum* etwas so aufgebaut ist – die KI war hier eher Lernpartner als Autor.

Mit dem Wissen, das ich seitdem aufgebaut habe, sehe ich inzwischen einiges, dass ich heute anders lösen würde – z. B. im Umgang mit Datenbankverbindungen oder der Trennung von UI- und Datenzugriffsschicht. Genau das macht dieses Projekt aber auch interessant als Schnappschuss eines bestimmten Lernstands.

## Funktionsumfang

- Erstellen, Bearbeiten und Löschen von Karteikarten
- Unterstützung für offene Fragen und Multiple-Choice-Karten
- Persistierung der Karten in einer MySQL-Datenbank
- Bedienung über eine Java Swing GUI

## Technologien

- Java (Swing für die GUI)
- MySQL (über JDBC)

## Status

Dieses Projekt wird nicht aktiv weiterentwickelt, sondern dient als **Lern-Schnappschuss**. Es ist Teil meiner persönlichen Dokumentation des Lernfortschritts im Bereich Java und OOP.
